# Great Awakening Map (starter)

## Run locally

1. Install Node.js (LTS)
2. In this folder:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

- Push this repo to GitHub
- Import the repo in Vercel
- Deploy (Vercel auto-detects Vite)

This project includes `vercel.json` so routes like `/topic/anunnaki` work on refresh.
